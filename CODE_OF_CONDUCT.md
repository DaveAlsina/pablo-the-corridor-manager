# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

**Positive behavior:**
- Being respectful and constructive in discussions
- Gracefully accepting feedback
- Focusing on what's best for the community

**Unacceptable behavior:**
- Harassment, trolling, or personal attacks
- Publishing private information of others
- Other conduct reasonably considered inappropriate

## Enforcement

Project maintainers will remove, edit, or reject contributions that violate this code. Repeated violations may result in a ban from contributing.

## Scope

This code applies to all project spaces (issues, pull requests, discussions).
